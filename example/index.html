 About TabMail

TabMail for Windows (95/98/Me/NT/2000/XP™) is mail user agent (MUA) for sending, receiving and managing your email messages. It's small and fast, conforms Internet Standards and supports Unicode characters.

Some other properties:

  • Simple view of compound messages
  • Message-thread can span many mailboxes
  • Automatic attachments compression
  • HTML messages view as plain text
  • Warn when open (or save) dangerous attachments
  • Easy PGP message encryption/decryption
  • SMTP and POP3 authentication
  • Support for SSL/TLS connections
  • Scanning attachments for viruses
  • Multi-user and multi-account
 Q&A

  1. Why I can't using AVG scanner with TabMail?

    Some AVG version(s) incorrect registers components. To avoid the problem add string ThreadingModel with value Apartment in HKEY_CLASSES_ROOT\CLSID\{67B30939-3B35-11d2-A595-002018648BA7}\InprocServer32 registry key.

  2. Can I see HTML messages in readable form?

    Yes. You can see HTML messages as plain text without graphics, applets, animations, banners etc. Check Preferences> Messages> Other> Show HTML messages as plain text.

  3. It is possible to make TabMail the default e-mail app?

    Yes. Open Internet Explorer Properties> Programs and choose TabMail from drop-down list.
